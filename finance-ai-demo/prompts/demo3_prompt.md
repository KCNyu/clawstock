# Demo 3 - 发票批量提取 Prompt

## 场景
每月收到大量发票，手工录入系统耗时，容易出错。

## 使用方法
把 `demo3_invoice_extract/` 文件夹放在当前目录，然后输入以下 prompt：

---

## Prompt（直接复制给 Claude Code）

```
demo3_invoice_extract 文件夹里有多张发票，请帮我：

1. 读取所有发票文件，提取以下字段：
   - 发票号码
   - 开票日期
   - 供应商名称
   - 金额（不含税）
   - 税额
   - 费用描述

2. 汇总成一张 Excel 表格 invoices_summary.xlsx

3. 在表格底部加一行合计：总金额、总税额

4. 检查是否有异常（金额特别大的、描述不清晰的），标注出来

5. 输出一个简短的录入完成报告
```

---

## 预期效果
7张发票 → 30秒内生成汇总表，手工录入至少需要 20-30 分钟。
这个 demo 视觉冲击力最强，建议放最后。
