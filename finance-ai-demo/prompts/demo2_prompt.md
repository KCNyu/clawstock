# Demo 2 - 多月数据合并分析 Prompt

## 场景
每月一个 CSV，季度汇报要手工合并、做对比，耗时费力。

## 使用方法
把 `demo2_merge_analysis/` 文件夹放在当前目录，然后输入以下 prompt：

---

## Prompt（直接复制给 Claude Code）

```
demo2_merge_analysis 文件夹里有1-3月的费用数据，请帮我：

1. 合并三个月的数据为一张完整表

2. 按部门统计每月总费用，计算环比变化：
   - 环比增长超过 20% 的部门重点标注

3. 找出费用最高的 Top 3 费用类型

4. 生成两个输出文件：
   - Q1_merged.xlsx：合并后的完整数据
   - Q1_report.xlsx：包含部门月度对比表、环比变化、Top3费用类型

5. 最后给我一段 3-5 句话的管理层摘要，结论先行
```

---

## 预期效果
Claude 自主完成数据合并、计算、生成报告，替代原本半天的手工工作。
