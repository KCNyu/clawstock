# Demo 1 - 数据清洗 Prompt

## 场景
财务报销表格格式混乱，手工整理要1-2小时。

## 使用方法
把 `demo1_clean_excel/messy_expenses.xlsx` 放在当前目录，然后输入以下 prompt：

---

## Prompt（直接复制给 Claude Code）

```
请读取 messy_expenses.xlsx，帮我做以下处理：

1. 清洗数据：
   - 统一日期格式为 YYYY-MM-DD
   - 清理金额列，去掉"元"字，统一为数字格式
   - 填充空白的部门字段（根据上下文推断或标记为"待确认"）
   - 去掉列名多余的空格

2. 分析：
   - 找出金额超过 5000 元的条目
   - 按部门汇总总费用

3. 输出：
   - 生成 cleaned_expenses.xlsx，超过5000元的行用黄色背景标注
   - 在第二个 sheet 里放部门汇总表
   - 打印一个简短的分析摘要，说明有哪些异常需要关注
```

---

## 预期效果
Claude 会自己写 Python 脚本，处理完输出两个文件，全程约 30-60 秒。
