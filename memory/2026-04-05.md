# 2026-04-05

- 发现并验证可用韩股查询链：Naver Finance polling API 可直接获取 KRX 实时/收盘数据。
- 可用接口：`https://polling.finance.naver.com/api/realtime/domestic/stock/<code>`
- 已验证代码：`000660`（SK海力士）、`005930`（三星电子）
- 返回字段包含：现价、涨跌额、涨跌幅、开高低、成交量、成交额、市值、市场状态、时间。
- 页面备用：`https://finance.naver.com/item/main.naver?code=<code>`
- 适用场景：韩股查询、判断港股映射ETF `07709` / `07747` 方向。
- 经验：当前服务器环境浏览器网关不可用时，优先走 Naver polling，不依赖 browser/CUA。
